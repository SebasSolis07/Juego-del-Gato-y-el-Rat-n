# Juego-del-Gato-y-el-Rat-n
Juego del Gato y el Ratón en una matriz cuadradada de dimensión 5. Consiste en un juego en el cual el gato es capaz de elegir el mejor movimiento posible para atrapar al raton, y este escapar del gato. Esta es la version donde el juego termina con el numero de intentos para escapar del raton. 
Lo interesante del codigo es que usamos recurvidad con su ciclo for
